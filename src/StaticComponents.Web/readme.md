# Static Components

Static Components is a one file ASP.NET Core library that allows you to write UI components while maintaining compatibility with Razor Pages and MVC.

- [Documentation](https://static-components.techgems.net/)
- [Nuget Package](https://www.nuget.org/packages/TechGems.StaticComponents)

## Previous versions

The nuget package for this library was previously called `TechGems.RazorComponentTagHelpers`.
